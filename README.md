# ginkoserver
To Ginkos Server follow these steps.


Start with Minecraft being installed.
Go to forge website and download the RECOMMENDED VERSION. Choose the INSTALLER option. http://files.minecraftforge.net/
Once finished load up Minecraft and you will now see a forge drop down here http://prntscr.com/s6han3
Next download the mods attached to this github they are all in one folder or here https://drive.google.com/file/d/1ypcCSy1QlZ5xJfHv8uldhrskKtQfTSLl/view
Now open minecraft with the forge version selected, and on the main menu select the MODS option.
Next in the bottom left choose OPEN MODS FOLDER to bring up where you will place the mods downloaded above.
Now reopen minecraft and run it on the forge version again and the mods will now be loaded.
Now ask to be whitelisted and for the ip in twitch chat!
